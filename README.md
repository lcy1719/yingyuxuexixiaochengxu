### 万字文档+英语学习小程序+Uniapp+SSM+Vue+sql文件+软件环境安装包
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、微信开发者工具/HBuilderX
技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML5（H5）、JavaScript、CSS、图片上传

系统功能模块：
本系统包含管理员和用户2个角色。
管理员角色：
- 个人中心：管理个人信息及设置。
- 用户管理：对平台用户进行增删改查等管理操作。
- 每日打卡管理：监控和管理用户的每日学习打卡情况。
- 备忘录管理：维护用户提交的备忘录信息。
- 学习计划管理：审核、调整用户的学习计划。
- 学习资源管理：上传、编辑或删除学习资源。
- 论坛交流管理：监管论坛内容，确保交流环境健康。
- 系统管理：进行系统设置、维护及升级，保障系统正常运行。
用户角色：
- 个人中心：查看和编辑个人资料，管理个人设置。
- 每日打卡：记录个人学习进度，参与连续学习挑战。
- 备忘录：创建和管理个人学习备忘，提醒学习任务。
- 学习计划：制定并追踪个人学习计划，促进学习自律。
- 学习资源：访问和利用平台提供的各类学习材料。
- 我的收藏：保存个人感兴趣的学习资源或帖子。
- 论坛交流：参与讨论，分享学习心得，与其他用户互动。
